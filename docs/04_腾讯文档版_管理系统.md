## 📐 八、计算公式汇总

### 达成率计算
```
达成率 = 实际值 / 目标值 × 100%
```

### 绩效得分计算规则
```
IF 达成率 ≥ 100%: 得分 = 100（封顶150%）
IF 80% ≤ 达成率 < 100%: 得分 = 达成率 × 100
IF 达成率 < 80%: 得分 = 0
```

### 工资计算
```
管理绩效实发 = 管理绩效基数 × 管理绩效得分 / 100
经营绩效实发 = IF(一票否决="是", 0, 经营绩效基数 × 经营绩效得分 / 100)
当月应发 = 固定工资 + 管理绩效实发 + 经营绩效实发 - 考勤扣减
```

### 合格率计算
```
合格率 = 合格数 / 检验数 × 100%
```
