## 1. 系统概述

### 1.1 系统简介

ACEMATIC运营管理系统是一个企业级战略执行管理平台，帮助您：
- 制定和跟踪公司战略
- 管理部门和个人工作计划
- 分配和执行任务
- 填报工作日志
- 接受绩效考核

### 1.2 角色说明

| 角色 | 权限范围 |
|------|----------|
| CEO | 全局管理、战略审批、人事任免 |
| 高管 | 分管部门管理、计划审批 |
| 经理 | 本部门管理、任务分配审核 |
| 员工 | 个人任务管理、日报周报 |

## 4. 战略管理

### 4.1 创建战略 (CEO/高管)

1. 点击左侧菜单"战略管理"
2. 点击右上角"新建战略"按钮
3. 填写战略信息：
   - 战略标题
   - 战略描述
   - 适用年度
   - 开始/结束日期
4. 点击"保存"

### 4.2 提交审核

1. 在战略列表找到草稿状态的战略
2. 点击"提交审核"按钮
3. 系统自动将状态改为"待审核"

### 4.3 审核战略 (CEO)

1. 找到待审核状态的战略
2. 点击"审核"按钮
3. 选择"通过"或"驳回"
4. 填写审核意见
5. 点击"确认"

### 4.4 状态说明

| 状态 | 说明 |
|------|------|
| 草稿 | 创建中，可编辑删除 |
| 待审核 | 等待审批 |
| 生效 | 审核通过，执行中 |
| 已完成 | 战略目标达成 |
| 已取消 | 战略被取消 |

## 5. 计划管理

### 5.1 创建计划

1. 点击左侧菜单"计划管理"
2. 点击"新建计划"按钮
3. 选择计划类型：
   - 公司计划 (CEO/高管)
   - 部门计划 (经理及以上)
   - 个人计划 (所有用户)
4. 填写计划信息
5. 保存为草稿

### 5.2 关联战略

计划可以关联公司战略，实现战略对齐：
1. 在创建/编辑计划时
2. 选择"关联战略"
3. 从下拉列表选择已生效的战略

### 5.3 计划审核流程

```
创建草稿 → 提交审核 → 等待审核 → 审核通过 → 生效执行
```

### 5.4 变更申请

已生效的计划需要修改时：
1. 点击计划操作按钮中的"申请变更"
2. 填写变更原因
3. 修改需要调整的内容
4. 提交变更申请
5. 等待原审核人审批

## 6. 任务管理

### 6.1 创建任务

1. 点击左侧菜单"任务管理"
2. 点击"新建任务"按钮
3. 填写任务信息：
   - 任务标题
   - 任务描述
   - 关联计划（可选）
   - 执行人
   - 优先级
   - 截止日期
4. 保存

### 6.2 任务状态

| 状态 | 说明 | 可执行操作 |
|------|------|------------|
| 草稿 | 创建中 | 编辑、删除、提交确认 |
| 待确认 | 等待执行人确认 | 确认、拒绝 |
| 进行中 | 执行中 | 更新进度、提交验收 |
| 已完成 | 等待验收 | - |
| 已验收 | 验收通过 | - |

### 6.3 更新任务进度

1. 找到进行中的任务
2. 点击"更新进度"按钮
3. 填写进度百分比
4. 记录工作内容
5. 保存

### 6.4 提交任务验收

1. 任务完成后点击"提交验收"
2. 填写完成说明
3. 提交给上级审核

## 12. 常见问题

### Q1: 忘记密码怎么办？
A: 请联系系统管理员重置密码。

### Q2: 为什么无法删除某些内容？
A: 只有草稿状态的内容可以删除，已生效的内容需先取消或申请变更。

### Q3: 任务无法提交验收？
A: 任务需要先标记为"已完成"状态才能提交验收。

### Q4: 如何修改已生效的计划？
A: 已生效的计划需要提交变更申请，经审批后才能修改。
