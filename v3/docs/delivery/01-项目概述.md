# ACEMATIC运营管理系统 - 项目概述

## 1. 项目背景

### 1.1 业务背景

随着企业规模扩大，传统的手工管理方式难以满足战略执行管控的需求：
- 战略目标分解困难，执行过程不透明
- 计划与任务脱节，无法追溯来源
- 考核评价缺乏数据支撑
- 部门协作效率低下

### 1.2 建设目标

建设一个**企业级战略执行管理平台**，实现：
- 从战略制定到执行考核的全流程数字化
- 任务来源追溯与战略对齐管理
- 精细化权限控制与审核工作流
- 实时数据看板与绩效分析

---

## 2. 项目范围

### 2.1 功能范围

| 功能模块 | 功能点 | 优先级 |
|----------|--------|--------|
| 认证授权 | 登录/登出/权限管理 | P0 |
| 战略管理 | 战略CRUD/审核工作流 | P0 |
| 计划管理 | 计划CRUD/审核/变更申请 | P0 |
| 任务管理 | 任务CRUD/分配/审核 | P0 |
| 执行管理 | 日志填报/进度更新 | P1 |
| 考核管理 | 绩效评分/结果查看 | P1 |
| 问题管理 | 问题登记/跟踪/闭环 | P1 |
| 报表管理 | 周报/报表生成 | P1 |
| 组织管理 | 部门/用户管理 | P0 |
| 系统设置 | 角色/权限配置 | P0 |

### 2.2 技术范围

- **前端**: React 19 + TypeScript + Vite + Tailwind CSS
- **后端**: Express.js + TypeScript + Prisma
- **数据库**: PostgreSQL
- **部署**: PM2 + Nginx
