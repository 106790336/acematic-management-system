## 1. 接口概述

### 1.1 基础信息

| 项目 | 说明 |
|------|------|
| 基础路径 | /api |
| 协议 | HTTP/HTTPS |
| 数据格式 | JSON |
| 认证方式 | Bearer Token (JWT) |

### 1.2 通用响应格式

```json
{
  "success": true/false,
  "data": { ... },
  "error": "错误信息"
}
```

### 1.3 认证头

```
Authorization: Bearer <token>
```

## 2. 认证接口

### 2.1 登录

**POST** `/api/auth/login`

**请求体**:
```json
{
  "username": "ceo",
  "password": "123456"
}
```

**响应**:
```json
{
  "success": true,
  "data": {
    "token": "eyJhbGci...",
    "user": {
      "id": "xxx",
      "username": "ceo",
      "name": "张总",
      "role": "ceo",
      "department": { "name": "总经办" }
    }
  }
}
```

### 2.2 获取个人信息

**GET** `/api/auth/profile`

**响应**:
```json
{
  "success": true,
  "data": {
    "id": "xxx",
    "username": "ceo",
    "name": "张总",
    "role": "ceo"
  }
}
```

### 2.3 登出

**POST** `/api/auth/logout`

## 3. 战略接口

### 3.1 获取战略列表

**GET** `/api/strategies`

**查询参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| status | string | 状态筛选 |
| year | number | 年度筛选 |
| page | number | 页码 |
| limit | number | 每页数量 |

**响应**:
```json
{
  "success": true,
  "data": {
    "items": [...],
    "total": 10,
    "page": 1,
    "limit": 20
  }
}
```

### 3.2 创建战略

**POST** `/api/strategies`

**权限**: `strategy:create`

**请求体**:
```json
{
  "title": "2026年度战略",
  "description": "战略描述",
  "year": 2026,
  "startDate": "2026-01-01",
  "endDate": "2026-12-31"
}
```

### 3.3 更新战略

**PUT** `/api/strategies/:id`

**权限**: `strategy:edit`

### 3.4 删除战略

**DELETE** `/api/strategies/:id`

**权限**: `strategy:delete`

### 3.5 提交审核

**POST** `/api/strategies/:id/submit`

### 3.6 审核战略

**POST** `/api/strategies/:id/review`

**权限**: CEO only

**请求体**:
```json
{
  "approved": true,
  "comment": "审核通过"
}
```
