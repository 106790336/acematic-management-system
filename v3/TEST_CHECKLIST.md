## 三、迭代阶段计划

### Phase 1: 核心闭环验证（本周）

**目标**: 确保「战略→计划→任务→子任务→审批」闭环可用

**测试重点**:
1. 登录后Dashboard加载
2. 部门/用户数据正确
3. 战略创建和状态流转
4. 子任务进度自动计算
5. CEO审批权限

**验收标准**: 
- 从创建战略到子任务验收的全流程走通
- 数据在各层级正确传递

### Phase 2: 数据导入与对齐（下周）

**目标**: 导入ACEMATIC真实组织数据

**任务**:
1. 创建28人组织架构（部门+岗位）
2. 导入年度目标（4000万营收）
3. 建立战略与部门KPI对齐
4. 历史数据迁移（如有）

### Phase 3: 移动端优化（第3周）

**目标**: 手机端可用

**重点**:
1. 响应式布局调整
2. 触屏交互优化
3. 移动端审批流程
4. 微信浏览器兼容
